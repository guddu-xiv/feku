Prayas One Student App - Split Files Exporter (GitHub Safe Plain-Text Edition)
========================================================================

Your database has grown large! This split package prevents file upload block errors, security alerts, and size limits on GitHub by exporting the dataset in plain-text (.txt) fragments rather than executable Javascript (.js) scripts.

FILES LIST IN THIS BUNDLE:
1. index.html                   - Main student testing portal.
2. config_part_1.txt, _2.txt... - Plain-text database fragments (pure base64 chunks, each under 5MB).
3. README.txt                   - This instruction file.

HOW TO DEPLOY TO GITHUB:
1. Extract/Unzip all the files in this ZIP archive.
2. Upload/Commit all files (including index.html and all 'config_part_*.txt' files) directly into your GitHub repository.
3. Make sure all the 'config_part_*.txt' files are placed in the SAME FOLDER/DIRECTORY as index.html.
4. Enable GitHub Pages in your Repository Settings pointing to this folder.

The application dynamically fetches and reassembles all segments into memory upon startup via secure browser fetches!
